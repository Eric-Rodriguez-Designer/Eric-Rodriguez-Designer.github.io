UCLA Alumni Scholars Club Web Application





Skills:


Web Design, Mobile Design, Prototyping, Wireframing, User Research


Project:


UCLA ATS has created some products to serve the needs of University Advancement Professionals. These products, such as Gift Agreement Builder, and Gift Hub have garnered interest from other Universities. In repsonse, my team was tasked with desiging and implementing a simple marketing site for these softwares to promote these products as a software as a service available from UCLA ATS.

This project was to create a modern web application to support the Alumni Affairs Alumni Scholars Club (ASC). This club supports UCLA Scholarship recipients with test banks, note sharing services, volunteer opportunities. Alumni Affairs came to ATS to assist with creating a modern web application to manage the logistics of this successful program. My team was responsible for the design of the application, while the building of the application was done with a third party vendor as well as UCLA develepors.

Mobile Mock-Ups



In our user research, one thing that became clear was that the primary way that students accessed the application was through their phones. To make sure that the application worked well in the mobile format, I created mobile mock-ups for all of the pages.







Components



Throughout the design process, there were a lot of changes and feedback. Following design best practices, we used Figma components for as many of the recurring web elements as possible. This way, if there was a change to a banner for instance, we were able to instantly update all instances of the component.
