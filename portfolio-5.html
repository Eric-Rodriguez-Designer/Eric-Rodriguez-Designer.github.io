UCLA Alumni Scholars Club Web Application

Role: UX/UI Designer 路 Skills: Web Design, Prototyping, Wireframing, User Research 路 Tools: Figma, Adobe XD 路 Timeline: 4 Weeks 路 Team: 4 Designers

Project Overview

The UCLA Alumni Scholars Club (ASC) supports scholarship recipients with academic and community resources like test banks, note sharing, and volunteer opportunities. Alumni Affairs partnered with our team to modernize the ASC platform and streamline logistics through a new responsive web application.

The Challenge

  • Difficult navigation for students
  • Poor mobile accessibility
  • Manual workflows for tracking volunteer hours
  • Outdated interface not aligned with UCLA's branding

Goals

  • Design a modern, responsive web app
  • Improve access to academic resources
  • Streamline event and volunteer coordination
  • Ensure accessibility and brand consistency

My Design Process

1. Discovery & Research

Conducted stakeholder interviews and gathered feedback from students and admins. Found major pain points in resource access, volunteer tracking, and mobile usability.

2. Wireframes & User Flows

Created wireframes that focused on clear navigation, simplified user journeys, and mobile-first design principles.

3. Visual Design & Prototyping

Developed a high-fidelity prototype using UCLA brand guidelines. Included modular layouts, clean dashboards, and accessibility-focused UI elements.

4. Testing & Handoff

Conducted usability testing with 7 users. Iterated on feedback and delivered annotated prototypes and style documentation to development teams.

Results

  • 40% faster task completion in user testing
  • Positive feedback on design clarity and mobile experience
  • Improved admin workflows with simplified dashboards
  • Fully responsive and accessible prototype delivered

Reflection

This project taught me how to balance brand requirements with user-centered design. I deepened my experience in collaborative design, accessibility, and building developer-ready prototypes.
